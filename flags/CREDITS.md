# Flag icons

Circular country-flag SVGs from **circle-flags** by HatScripts.

- Source: https://github.com/HatScripts/circle-flags
- Commit: 81c3c5e73340ede011b9e958a3c52e5c6f86459e
- License: MIT (see below)

Vendored (self-hosted) so the page makes **zero external requests** at runtime,
consistent with the fonts in `fonts/`. Files are named by ISO 3166-1 alpha-2
(e.g. `br.svg`), plus `gb-eng`/`gb-sct`/`gb-wls`. `app.js` derives the filename
from each team's flag emoji.
